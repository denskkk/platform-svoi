# Розумний пошук послуг

## Як це працює

Система пошуку автоматично розширює ваш запит синонімами та спорідненими термінами, щоб знайти більше релевантних послуг.

## Приклади

### Будівництво і ремонт
- **"сантехнік"** знайде також: прораб, будівельник, ремонт, майстер, монтаж
- **"електрик"** знайде також: прораб, будівельник, електромонтаж, проводка
- **"прораб"** знайде також: будівельник, ремонт, сантехнік, електрик, оздоблення

### Авто
- **"автомеханік"** знайде також: СТО, ремонт авто, діагностика, автосервіс
- **"шиномонтаж"** знайде також: СТО, автосервіс, колеса, шини

### Краса
- **"перукар"** знайде також: стиліст, салон краси, барбер, стрижка
- **"манікюр"** знайде також: нігті, педикюр, салон краси, майстер манікюру

### IT
- **"програміст"** знайде також: розробник, веб, сайт, додаток, IT
- **"дизайнер"** знайде також: веб, графіка, логотип, брендинг

## Автоматичне визначення категорії

Якщо ви не вказали категорію, система спробує визначити її автоматично:
- "сантехнік" → категорія "Ремонт"
- "перукар" → категорія "Краса"
- "програміст" → категорія "IT"

## Сортування результатів

1. Послуги з прямим збігом (наприклад, "сантехнік" у назві) — найвище
2. Послуги зі спорідненими термінами (наприклад, "прораб" для запиту "сантехнік")
3. Серед однакової релевантності — за рейтингом виконавця

## Додавання нових синонімів

Файл: `src/lib/searchKeywords.ts`

Щоб додати нові синоніми, відредагуйте об'єкт `SEARCH_SYNONYMS`:

```typescript
export const SEARCH_SYNONYMS: Record<string, string[]> = {
  'ваше_слово': ['синонім1', 'синонім2', 'споріднена_категорія'],
}
```

## Приклади використання

### У коді:
```typescript
import { expandSearchQuery, detectCategoryFromQuery } from '@/lib/searchKeywords'

const terms = expandSearchQuery('сантехнік')
// ['сантехнік', 'прораб', 'будівельник', 'ремонт', 'майстер', ...]

const category = detectCategoryFromQuery('електрик')
// 'remont'
```

### У формі пошуку:
1. Введіть будь-яке слово: "сантехнік"
2. Система автоматично знайде всі схожі послуги
3. Результати відсортовані за релевантністю
